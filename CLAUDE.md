# Project guidance for Claude Code

## Commit policy

- Commits are authored solely by the developer currently logged in (`git config user.name` / `user.email`). Do not change the author.
- **Never** add a `Co-Authored-By: Claude` trailer.
- **Never** add the "🤖 Generated with Claude Code" line, the Claude icon/emoji, or any other Claude/Anthropic attribution to commit messages or PR descriptions.
- Write commit messages as the developer would — no AI co-author credit of any kind.
